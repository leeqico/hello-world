> 对于一个大型的项目，推荐使用 Vue CLI 来搭建我们的整个环境工程，通过这种方式，我们可以最大限度发挥 Vue 的能力，提升我们的开发效率，也利于我们项目的后期维护。

## Vue CLI
```
npm install –g @vue/cli
vue create hello-world
cd hello-world
npm run serve
```

## Project setup
```
npm install
```
